# VisionQuest
VisionQuest — профессиональное Android-приложение-викторина на Java с архитектурой MVС. Реализована загрузка и парсинг JSON-данных, обработка событий UI, динамическое обновление интерфейса. Код соответствует принципам Clean Code, масштабируем, легко расширяется. Минимальная версия — Android 5.0 (API 21).
